# NotebookLM クローン - YouTuber向け動画制作支援ツール

## 概要

これは、日本人のYouTuberが海外のYouTubeを参考にして動画を作成するための支援ツールです。GoogleのNotebookLMのUIをベースに設計されており、スタイル（ノウハウ）の収集、ソース（参考動画）の管理、そして動画シナリオの作成を一元的に行うことができます。

## 主な機能

### 1. スタイル管理（左サイドバー）
- **目的**: 参考にしたYouTuberの動画の雰囲気、話し方、編集テクニックなどのノウハウを記録
- **機能**:
  - 「ソースを追加」: YouTube動画をAI分析してスタイルとして保存
  - 「ファイルを追加」: Markdownファイルを直接追加
  - タイムスタンプ形式のファイル名（例: `2026年01月20日15.123.md`）

### 2. ソース管理（左サイドバー）
- **目的**: 参考にした動画や資料を管理
- **機能**:
  - 「ソースを追加」: YouTube動画URLを入力してAI分析
  - 「ファイルを追加」: PDF、テキスト、Markdownなどの資料を追加
  - 動画の要約、ポイントを自動記録

### 3. AIディスカッション（中央エリア）
- **目的**: 収集したスタイルやソースについてAIと対話
- **機能**:
  - AIチャットで動画の分析、要約、質問回答
  - 複数のソースを組み合わせた分析が可能

### 4. 動画シナリオ作成（右サイドバー）
- **目的**: スタイル、ソース、ディスカッションを元に動画シナリオを作成
- **機能**:
  - AIによる自動シナリオ生成
  - 収集した情報を統合したシナリオの執筆
  - Markdown形式で編集・保存

### 5. エディタ（選択時のみ表示）
- **機能**:
  - スタイル、ソース、シナリオをクリックすると展開
  - Markdown形式で編集可能
  - 自動保存機能
  - Markdownファイルとしてエクスポート
  - YouTube動画の埋め込み表示（URLがある場合）

## ワークフロー

1. **スタイルを収集**: お気に入りのYouTuberの動画を分析して、編集テクニックや話し方のノウハウを記録
2. **ソースを追加**: 参考にしたい動画URLや資料を追加
3. **AIと対話**: チャットで動画の内容を分析、質問、要約
4. **シナリオ作成**: 収集した情報を元に、オリジナルの動画シナリオを作成

## 技術スタック

- **フロントエンド**: Next.js 16.1.3, React 19.2.3
- **スタイリング**: Tailwind CSS 4
- **言語**: TypeScript
- **バックエンド**: Next.js API Routes
- **データベース**: Supabase (PostgreSQL)
- **ORM**: Prisma

## CI/CD

### GitHub Actions

このプロジェクトではGitHub Actionsを使用して、mainブランチへのマージ時に自動的にデータベーススキーマを更新します。

**ワークフロー**:
- `.github/workflows/prisma-migrate-production.yml`: mainブランチへのマージ時にPrismaマイグレーションを本番DBに適用

**詳細な設定方法**: [docs/github-actions-setup.md](docs/github-actions-setup.md)

**必要な設定**:
1. GitHub Secretsに `DATABASE_URL`（本番DB接続文字列）を設定
2. PRをマージすると自動的にマイグレーションが適用されます

## 開発

```bash
cd frontend
bun install
bun run dev
```

ブラウザで http://localhost:3000 にアクセスしてください。

## プロダクトの特徴

- **NotebookLM風UI**: 見慣れた3カラムレイアウト
- **ファイル蓄積型**: すべての情報をMarkdownファイルとして保存
- **AI統合**: YouTube動画の自動分析、要約、対話
- **YouTuber特化**: 動画制作に特化した機能設計
